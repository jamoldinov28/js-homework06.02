JavaScriptning Loop mavzusi

Loop takrorlash degan manoni bildiradi

1. loopda to'g'ri qatorli sonlarga misol keltiraman

for(i = 1; i < 10; i++){
console.log(i);
}

2. loopda teskari qatorli sonlarga misol keltiraman

for(i = 10; i > 0; i--){
console.log(i);
}

JavaScriptning Loop (While, do-While) mavzusi

1.While va for ning farqi sintaksizida farqi bor xolos

Whileda misol keltiraman

Let i = 6
While(i = 6){
i++
console.log("While ", i )
}

2.do-While shartni 1chi tekshirgani uchun xato bolsa xam 1 martta ishlaydi

do-Whileda misol keltiraman

Let i = 6
do{
console.log("do While ", i )
i++
}While(i = 6)
